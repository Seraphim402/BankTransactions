# Банковские транзакции

Учебное приложение на C# + SQLite:

- база данных SQLite;
- серверная часть: консольное приложение;
- клиентская часть: WPF-окно, можно запускать несколько экземпляров;
- unit-тесты для бизнес-логики.

## Запуск

### Через готовые exe

После публикации приложения готовые файлы лежат в папке `dist`:

- `dist\Server\BankTransactions.Server.exe` - сервер;
- `dist\Client\BankTransactions.Client.exe` - оконный клиент;
- `dist\Запустить сервер.bat` - быстрый запуск сервера;
- `dist\Запустить клиент.bat` - быстрый запуск клиента.

Сначала запустите сервер, затем один или несколько клиентов.

### Через dotnet

1. Запустить сервер:

```powershell
dotnet run --project .\src\BankTransactions.Server\BankTransactions.Server.csproj
```

2. Запустить один или несколько клиентов:

```powershell
dotnet run --project .\src\BankTransactions.Client\BankTransactions.Client.csproj
```

3. Войти в клиенте:

- логин: `admin`
- пароль: `admin`

## Сценарий для отчета

1. Запустить сервер. В консоли видно количество подключенных окон-клиентов.
2. Запустить два окна клиента.
3. В первом окне войти под `admin/admin`.
4. Создать физическое лицо и юридическое лицо.
5. Создать зарплатный счет физического лица и валютный счет юридического лица.
6. Пополнить зарплатный счет.
7. Выполнить перевод на валютный счет через зарубежный банк-партнер.
8. Проверить список транзакций: сумма, комиссия, банк-партнер, дата.
9. Удалить клиента: операция доступна только после входа.

## Состав БД

- `Clients` - физические и юридические лица;
- `Accounts` - зарплатные, валютные, накопительные счета;
- `Transactions` - пополнения и переводы;
- `Commissions` - правила комиссий;
- `PartnerBanks` - банки-партнеры;
- `Users` - пользователи для входа.
